# Bellhaven Hotel Booking System

Bellhaven is a premium full-stack Hotel Booking System developed using React.js and Spring Boot. It provides a seamless hotel reservation experience with secure authentication, room booking, an admin dashboard, and complimentary cab services for guests.

---

## Features

### User Features

- User Registration and Login
- Secure Authentication using Spring Security
- Browse Available Rooms
- View Room Details and Amenities
- Hotel Room Booking System
- Complimentary Cab Service for Hotel Guests
- View Booking Details
- Responsive and Modern User Interface
- Contact and Reviews Section

### Admin Features

- Admin Login
- Admin Dashboard
- Manage Hotel Rooms
- Manage Bookings
- View Customer Details
- Monitor Room Availability
- Booking Management System

---

## Complimentary Cab Service

Bellhaven provides a free cab service for guests who book rooms through the platform.

### Cab Service Includes

- Complimentary Pickup Service
- Driver Details
- Cab Tracking Information
- Pickup Notifications
- Secure Transportation for Guests

## Frontend Setup

### Navigate to Frontend Folder

```bash
cd bellhaven-frontend
```

### Install Dependencies

```bash
npm install
```

### Run the Frontend

```bash
npm start
```

or

```bash
npm run dev
```

---

## Backend Setup

### Navigate to Backend Folder

```bash
cd bellhaven-backend
```

### Install Dependencies

```bash
mvn clean install
```

### Run the Spring Boot Application

```bash
mvn spring-boot:run
```

or run:

```bash
BellhavenBackendApplication.java
```
